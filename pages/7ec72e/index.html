# 像大多数编程语言一样，Pthon保留了一些特殊字符用来操作。

# 1.算术操作符

操作符 作用 示例
+ 加法-两个数相加 a+b
– 减法-两个数相减 a-b
* 乘法-两个数的乘积 a*b
/ 除法-两个数的商 a/b
// 除法-抛去小数部分，只保留整数 a//b
% 取模-保留余数 a%b
** 指数-a的b次方 a**b
a=7
b=4

print('Sum : ', a+b)
print('Subtraction : ', a-b)
print('Multiplication : ', a*b)
print('Division (float) : ', a/b)
print('Division (floor) : ', a//b)
print('Modulus : ', a%b)
print('Exponent : ', a**b)
