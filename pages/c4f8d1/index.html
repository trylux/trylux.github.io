# 4.字节(Bytes)

# 在Python里字节是一个不变的类型。

# (1)Byte对象是包含多个字节的对象，字符串是包含多个字符的对象

# (2)字节是机器可读的对象，字符串是人类可读的对象

# (3)字节因为是机器可读的，所以可以直接存储到硬盘，而字符串在读写硬盘前需要编码

>>> # Make an empty bytes object (8-bit bytes)
