# 4.格式化

# (1)转义字符

转义字符 实际输出
\\ 反斜杠
\" 双引号
\a 响铃
\b 退格
\000 空
\n 换行
\v 纵向制表符
\t 横向制表符
\f 换页
\r 回车
\yyy 八进制数，y 代表 0~7 的字符，例如：\012 代表换行
\xyy 十六进制数，以 \x 开头，y 代表的字符，例如：\x0a 代表换行
\other 其它的字符以普通格式输出

# (2)格式输出

print ("Employee Name: %s,\nEmployee Age:%d" % ('Ashish',25))

#输出: Employee Name: Ashish, 
#输出: Employee Age: 25
1
2
3
4
符号 描述
%c 格式化字符及其ASCII码
%s 格式化字符串
%d 格式化整数
%u 格式化无符号整型
%o 格式化无符号八进制数
%x 格式化无符号十六进制数
%X 格式化无符号十六进制数（大写）
%f 格式化浮点数字，可指定小数点后的精度
%e 用科学计数法格式化浮点数
%E 作用同%e，用科学计数法格式化浮点数
%g %f和%e的简写
%G %f 和 %E 的简写
%p 用十六进制数格式化变量的地址
